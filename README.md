# Calculator App

A simple calculator application built using Python and Tkinter.

## Features

- Basic arithmetic operations (addition, subtraction, multiplication, division).
- Operand highlighting for improved user experience.
- Error handling for invalid expressions.
- Calculation history with the ability to recall and reuse previous results.

## Usage

1. Run the calculator application:

   ```bash
   python calculator.py
   ```

2. Use the buttons to input numbers and perform calculations.

3. Operand highlighting makes it clear which part of the expression is being modified.

4. Check the calculation history in the side panel and scroll through previous results.

## Requirements

- Python 3.x
- Tkinter (usually included with Python installations)
